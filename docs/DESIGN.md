The project follows a **simple retro pixel art** style inspired by classic strategy games. Assets should prioritize **clarity and readability** over visual complexity.

## Asset Resolution

* Characters: **32×32 px**
* Tiles: **32×32 px**
* UI Icons: **16×16 px**
* Small Objects: **16×16 px**
* Backgrounds: **1920×1080 px**

Use fixed pixel sizes whenever possible.

## Perspective

* Top-down with a slight angle.
* Consistent perspective across all assets.

## Colors

Use a limited color palette with high contrast.

### Primary Colors

| Purpose | RGB |
|---------|-------------|
| Background | **RGB(32, 36, 44)** |
| Panel | **RGB(52, 58, 64)** |
| Border | **RGB(96, 82, 60)** |
| Primary Accent | **RGB(212, 175, 55)** |
| Secondary Accent | **RGB(120, 170, 255)** |
| Danger | **RGB(196, 58, 58)** |
| Success | **RGB(72, 176, 97)** |

### Text Colors

| Type | RGB |
|------|-------------|
| Primary Text | **RGB(245, 240, 220)** |
| Secondary Text | **RGB(180, 180, 180)** |
| Disabled Text | **RGB(120, 120, 120)** |
| Highlight | **RGB(255, 220, 90)** |

Avoid pure black (`0,0,0`) and pure white (`255,255,255`) whenever possible.

## Outlines

* Use dark outlines to improve readability.
* Avoid pure black unless necessary.

## Animations

Keep animations short and responsive.

* Idle: **2–4 frames**
* Walk: **4–6 frames**
* Attack: **4–8 frames**
* Effects: **3–6 frames**

## General Guidelines

* Keep sprites simple and recognizable.
* Avoid excessive detail.
* Maintain a consistent scale between all assets.
* Design assets for gameplay first, aesthetics second.
* Every new asset should match the existing visual style.
